# 💼 Currículo Online

Este é o repositório do meu **currículo digital**, criado com **HTML** e **CSS** para apresentar minhas informações profissionais de forma organizada e acessível.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
